# 🛒 Tienda Galindez  

> 🌟 Donde comprar es una experiencia feliz  

Bienvenido a **Tienda Galindez**, una aplicación web tipo e-commerce desarrollada con **Node.js, Express y MongoDB**.  

Este proyecto no solo gestiona productos (CRUD), sino que busca ofrecer una experiencia cercana, confiable y agradable para el usuario 🤝  

---

## 🐾 Nuestra esencia: El Quokka  

Nuestra identidad está inspirada en el **quokka**, conocido como *el animal más feliz del mundo* 😄  

💡 ¿Qué representa para nosotros?  
- 😊 Sonrisa → Clientes felices  
- 🤝 Cercanía → Atención amable y humana  
- 💚 Confianza → Experiencia positiva al comprar  

En **Tienda Galindez** creemos que cada interacción debe sentirse así:  
**simple, amigable y con buena vibra ✨**  

---

## 🚀 Tecnologías utilizadas  

- ⚙️ Node.js  
- 🚏 Express  
- 🍃 MongoDB  
- 🎨 EJS (plantillas dinámicas)  
- 💡 Arquitectura MVC  

---

## 📦 Funcionalidades principales  

✔ Gestión de productos (CRUD)  
✔ Manejo de categorías y marcas  
✔ Renderizado dinámico con EJS  
✔ Estructura escalable tipo e-commerce  
✔ Integración con base de datos
